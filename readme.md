# Selwyn.cc
My personal website.

## General Prerequisites
Node.js, [latest LTS is recommended](https://nodejs.org/en/about/releases/).

## Quick start

### Setup
```sh
git clone git@github.com:Siilwyn/selwyn.cc.git
cd selwyn.cc
npm ci
```

### Running
```sh
npm run dev
```

## Production

### Running
```sh
npm run build
npm run server
```
