# Selwyn.cc
My personal website.

## General Prerequisites
Node.js, [latest LTS is recommended](https://nodejs.org/en/about/releases/).

## Quick start

### Setup
Clone the repository, and install dependencies from the project root with:
```sh
npm ci --ignore-scripts
```

### Running
```sh
npm run dev
```

## Production

### Running
```sh
npm run build
npm run server
```
